# Churn Rate Prediction (R)

This project focuses on predicting customer churn using Spotify-related data.

## Objective
Build a predictive model to identify users likely to churn.

## Tools & Technologies
- R
- Data preprocessing
- Supervised and Unsupervised models

## Files
- SpotifyChurnRate_Group8.R: main analysis and modeling script
